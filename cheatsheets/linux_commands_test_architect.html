Linux Commands for Test Architect Interview

1. File & Directory Management

CommandDescription
ls -lList files in long format
ls -aList all files including hidden
pwdPrint current directory path
cd /path/to/dirChange directory
mkdir <dir>Create directory
rm <file>Delete a file
rm -rf <dir>Delete directory recursively
touch <file>Create an empty file
cp source destCopy file or directory
mv source destMove or rename file/directory

2. File Viewing & Editing

CommandDescription
cat fileDisplay file content
less fileView file content page by page
tail -f fileView file live (log monitoring)
head -n 10 fileView first 10 lines
nano file / vi fileEdit file from terminal
grep 'text' fileSearch text inside files
grep -r 'text' dir/Recursive search
wc -l fileCount lines in file

3. File Permissions & Ownership

CommandDescription
ls -lCheck permissions of files/directories
chmod 755 fileChange file permissions
chown user:group fileChange file owner/group
umaskDefault permission for new files

4. Process & System Management

CommandDescription
ps -efList running processes
top / htopMonitor live system/process usage
kill <PID>Kill a process
kill -9 <PID>Force kill process
df -hDisk usage info
du -sh <dir>Directory size
free -mMemory usage in MB

5. Networking & Connectivity

CommandDescription
ping hostCheck connectivity to host
curl http://urlTest API / HTTP request
wget http://urlDownload file
netstat -tulnCheck open ports
ss -tulnpList listening ports with processes
telnet host portTest connection to a port

6. Archiving & Compression

CommandDescription
tar -cvf archive.tar dir/Create tar archive
tar -xvf archive.tarExtract tar archive
gzip fileCompress file
gunzip file.gzDecompress file
zip file.zip dir/Zip directory
unzip file.zipUnzip file

7. Logs & Monitoring

CommandDescription
tail -f /var/log/syslogMonitor system logs
grep ERROR /var/log/app.logFind error logs
watch 'command'Run command repeatedly (e.g., monitor logs)
dmesgKernel logs, useful for debugging

8. Package Management

CommandDescription
apt-get updateUpdate package list (Debian/Ubuntu)
apt-get install <pkg>Install package
yum install <pkg>Install package (RedHat/CentOS)
dpkg -lList installed packages
rpm -qaList installed packages (RPM)

9. Environment & Variables

CommandDescription
echo $PATHPrint environment variable
export VAR=valueSet environment variable
printenvList all environment variables
envView current environment
source ~/.bashrcReload bash profile settings

10. Automation / CI Commands

  • cron / crontab -e → Schedule jobs
  • chmod +x script.sh → Make script executable
  • ./script.sh → Run shell script
  • ssh user@host → Remote login
  • scp file user@host:/path → Copy files to remote
  • rsync -av source/ dest/ → Sync directories
