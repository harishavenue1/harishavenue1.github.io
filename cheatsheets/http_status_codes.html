HTTP Status Codes — Complete Cheat Sheet

A compact reference of HTTP status codes, grouped by category. Use this while building or testing APIs.

1xx — Informational

CodeDescription
100Continue — client should continue with request
101Switching Protocols — server is switching protocols as requested
102Processing (WebDAV) — server has received and is processing the request
103Early Hints — used to return preloads and hints to client

2xx — Success

200OK — request succeeded
201Created — resource created (typically after POST)
202Accepted — request accepted for processing (async)
203Non-Authoritative Information — returned meta-information not from origin server
204No Content — success but no body to return
205Reset Content — instructs client to reset view (no body)
206Partial Content — server is delivering partial content (range requests)
207Multi-Status (WebDAV)
208Already Reported (WebDAV)
226IM Used — response is a representation of the result of one or more instance-manipulations

3xx — Redirection

300Multiple Choices — multiple representations available
301Moved Permanently — resource moved to new URL
302Found — temporary redirect
303See Other — redirect to GET on different URI
304Not Modified — cached resource valid, no need to resend
305Use Proxy — deprecated / rarely used
306(Unused)
307Temporary Redirect — keeps the same HTTP method
308Permanent Redirect — keeps the same HTTP method

4xx — Client Errors

400Bad Request — request malformed or invalid
401Unauthorized — authentication required or failed
402Payment Required — reserved for future use
403Forbidden — authenticated but not allowed
404Not Found — resource doesn't exist
405Method Not Allowed — HTTP method not supported by endpoint
406Not Acceptable — content negotiation failed
407Proxy Authentication Required
408Request Timeout — server timed out waiting for the request
409Conflict — request conflicts with current server state
410Gone — resource permanently removed
411Length Required — Content-Length header required
412Precondition Failed — one or more preconditions failed
413Payload Too Large — request entity too large
414URI Too Long — request-URI is too long
415Unsupported Media Type — media type not supported
416Range Not Satisfiable — invalid range header
417Expectation Failed
418I'm a teapot — Easter egg (RFC 2324)
421Misdirected Request — request sent to a server that can't produce a response
422Unprocessable Entity (WebDAV) — semantic errors in request
423Locked (WebDAV)
424Failed Dependency (WebDAV)
425Too Early — server is unwilling to risk processing a request that might be replayed
426Upgrade Required
428Precondition Required
429Too Many Requests — rate limiting
431Request Header Fields Too Large
451Unavailable For Legal Reasons

5xx — Server Errors

500Internal Server Error — generic server error
501Not Implemented — server does not support requested functionality
502Bad Gateway — invalid response from upstream server
503Service Unavailable — server is overloaded or down for maintenance
504Gateway Timeout — upstream server failed to respond in time
505HTTP Version Not Supported
506Variant Also Negotiates
507Insufficient Storage (WebDAV)
508Loop Detected (WebDAV)
510Not Extended
511Network Authentication Required

Reference: Common HTTP status codes (RFCs and commonly used extensions). This sheet includes standard, WebDAV and some extension codes.
